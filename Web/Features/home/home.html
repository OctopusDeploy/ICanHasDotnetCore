Can I port my application to .NET Core?

It really depends on what you mean by .NET Core. Most commonly the term is used to refer to the .NET Standard Librarylaunch (.NET Standard). However it may also refer to the new tool stack. For more information, check out the FAQ for more information.

There are two main aspects to determining whether you project can be ported to .NET Standard:

  1. Are any .NET BCL types and methods used that are not part of the standard. Michael Whelan has written an excellent guidelaunch on analysing an existing project and porting it.
  2. Have your other dependencies been ported to .NET Standard. To do that, either upload your project's package.config files below or download the command line tool.
